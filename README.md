# MetalDesign - Modern Metal Construction

A premium React landing page for a metal construction company, featuring 3D visualizations, smooth animations, and a modern industrial aesthetic.

## 🚀 Features

- **3D Hero Section:** Interactive 3D scene built with React Three Fiber.
- **Modern UI:** Styled with Tailwind CSS for a sleek, industrial look.
- **Scroll Animations:** Powered by Framer Motion for a dynamic user experience.
- **Responsive Design:** Optimized for all devices from mobile to desktop.
- **Component Based:** Clean architecture with reusable React components.

## 🛠️ Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS 4
- **3D Engine:** Three.js / React Three Fiber / Drei
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 📦 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mdesdoua-creator/Projet_ConstructionMetal.git
    ```
2.  **Install dependencies:**
    ```bash
    cd Projet_ConstructionMetal
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
